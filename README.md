## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)

### Running the Project

1. Open `index.html` in your web browser by:
   - Double-clicking the file, or
   - Right-clicking and selecting "Open with" your preferred browser, or
   - Using the VS Code Live Server extension (recommended for development)

### Using Live Server (Recommended)

If you have the Live Server extension installed in VS Code:

1. Right-click on `index.html`
2. Select "Open with Live Server"
3. The page will open in your default browser with auto-reload on save
